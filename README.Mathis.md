# Projet AlgoBox — Travail en binôme

## Description
Projet AlgoBox collaboratif utilisant GitHub pour le suivi des versions.
Les fichiers `.alg` sont stockés dans le dossier `src/`.

## Structure
```
projet_algobox/
├─ src/
│  └─ algoboxx.alg
└─ README.md
```

## Prérequis
- AlgoBox installé
- Git
- Un compte GitHub

## Ouvrir / Exécuter
1. Cloner le dépôt :
   ```bash
   git clone <URL_DU_DEPOT>
   ```
2. Ouvrir AlgoBox
3. Charger le fichier :
   - `src/algoboxx.alg`

## Workflow conseillé
```bash
git pull origin main
# modifier les fichiers .alg dans src/
git add src/*.alg
git commit -m "Description courte et précise"
git push origin main
```

## Lien GitHub
À compléter et figer avec un tag/release pour le rendu.
